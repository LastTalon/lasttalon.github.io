Polygon Editor

Date: 2012

Contribution: Developed solo

Description: This is a simple polygon editor. It draws a polygon to the screen that the user can move around and manipulate. This project was one of my earlier object oriented programming projects as well as one of my first graphical applications. I made this after having taken an independent study in Java during my senior year of high school. I did this as a sort of test project to see what I was able to do at the time and it still holds up pretty well.

Unity Pong

Date: 2015

Contribution: Developed solo

Description: This is a version of the classic Pong, but made in Unity. It is a relatively simple game, closely resembling Pong and air hockey. The game has two modes, one that uses Unity physics for movement, and one that uses programmed movement. This is one of my first projects done in Unity, done shortly after some initial tutorials. It was just meant as a way to get used to development and the way things work in Unity. Although this project was fairly simple it really helped me learn what I was doing with Unity and I feel confident in my ability to use Unity now.
